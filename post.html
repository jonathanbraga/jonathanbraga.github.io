Vamos começar!

O post de hoje vai ser uma breve introdução ao Xamarin Forms e no final vamos criar o nosso primeiro app. O mundo mudou quando a Microsoft trouxe a Xamarin para a realidade open source, trouxe a facilidade na construção de apps. Agora não se é mais necessário 3 times de programadores ou um super programador que saiba as 3 linguagens para desenvolver apps para IOS, Android e WinPhone. Hoje, temos o Xamarin Forms que proporciona o compartilhamento de código e assim a construção de apps nativos!

Xamarin Forms, proporciona todo o conforto da linguagem C# e ainda nos permite usar API nativas de cada plataforma.
